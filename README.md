## ✅ 待辦清單 (Todo List)

### 專案介紹

這是一個使用 **React** 製作的簡單待辦清單，主要用來練習元件拆分、狀態管理與事件處理。  
使用者可以新增、刪除、完成或取消完成待辦事項。

### 功能特色

- ➕ 新增待辦事項
- 🗑️ 刪除待辦事項
- ✔️ 切換完成/未完成狀態
- 📋 清單顯示目前的待辦與完成項目

### 技術棧

- **React** - 元件化開發
- **JavaScript (ES6+)** - 邏輯處理
- **CSS / Tailwind**（可依實作情況更改）
